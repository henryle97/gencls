## Introduction 
GenCLS is an open source image classification toolbox based on PyTorch. 

### Majoir features

- Various backbones 
- Bag of training tricks
- Intergrate model compression: Prunning, Quantization, KD
- Intergrate model export format: ONNX, TensorRT
